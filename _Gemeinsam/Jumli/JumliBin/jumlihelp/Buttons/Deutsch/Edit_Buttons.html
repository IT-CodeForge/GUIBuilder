Edit Buttons



Button Compilieren

Mit dem Button Compile erzeugt Jumli aus Ihrem Sourcecode im Edit-Fenster eine Datei, die der Computer verstehen kann. Sind im Sourcecode Fehler enthalten, so werden diese im Message-Fenster angezeigt.



Button Build

Dieser Button funktioniert im Prinzip genauso wie der Compile-Button, nur das dieses Mal alle vorhandenen Sourcecode-Dateien im Projekt compiliert werden und nicht nur der Code, der im Edit-View angezeigt wird. Als Ergebnis erhalten sie das ausführbare Programm, an dem Sie arbeiten.



Button Ausführen

Über den Ausführen-Button können Sie eine ausführbare Datei laufen lassen. Ist die ausgewählte Datei nicht ausführbar, so werden Sie darauf hingewiesen. Auch wenn Sie vergessen haben die Datei zu compilieren, weist Jumli Sie darauf hin.



Button Debugger

Mit diesem Button springen Sie in den Debug-Modus. Hier können Sie einen Programmierfehler leichter finden. Näheres wird unter den Debug-Buttons erklärt.
