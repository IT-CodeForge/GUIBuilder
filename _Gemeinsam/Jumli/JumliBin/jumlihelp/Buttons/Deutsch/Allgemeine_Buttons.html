Allgemeine Buttons



Button Neu

Über diesen Button kommen Sie in ein Dialog-Fenster, in dem Sie wählen können, ob Sie ein neues Projekt öffnen. Einem bereits geöffneten Projekt können Sie auch eine neue Datei hinzufügen (Hier haben Sie die Auswahl zwischen einer Sourcecode-Datei oder einer HTML-Seite). Haben Sie vorher in einem anderen Projekt gearbeitet und nicht abgespeichert, werden Sie darauf aufmerksam gemacht.



Button Öffnen

Über den Button Öffnen können Sie ein bestehendes Projekt öffnen oder andere Dateien zu dem aktuellen Projekt hinzufügen.



Button Speichern

Über diesen Button können Sie Ihre bisherige Arbeit abspeichern. Wollen Sie ein anderes Projekt öffnen und haben vergessen vorher zu speichern, dann werden Sie von Jumli darauf aufmerksam gemacht.



Button Fenster wechseln

Dieser Button dient dem Wechsel der Ansichten zwischen Edit-Fenster und Modell-Fenster.
