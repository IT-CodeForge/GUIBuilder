Rechtliche Hinweise

Die Jumli Installation

Die Installation von Jumli ist nicht schwer. Zunächst muss Jumli natürlich von www.jumli.de heruntergeladen werden. Man erhält hier für Windows eine ".exe"-Datei. Mit einem Doppelklick auf "setup.exe" starten Sie die Installation. Folgen Sie nun den Installationshinweisen. Im Startmenü wird ein Icon erstellt, über welches sich Jumli nach der Installation starten lässt.



Die Installation des Java-SDK


Für die Programmierung in Java mit Jumli ist die Installation des J2SE-SDK, also des "Java 2 Platform, Standard Edition Software Developement Kit" (kurz JDK) notwendig. Dieses wird Ihnen auf der Hompage von Sun Microsystems zum kostenlosen Download angeboten. Für den Inhalt dieser Seiten ist Jumli.de nicht verantwortlich!

Sie erhalten eine ".exe"-Datei, die nach einem Doppelklick das komplette SDK installiert. Wir empfehlen das SDK in Ordnern wie z.B. c:\jdk1.4 oder c:\Programme\jdk1.4 zu installieren. Dies erleichtert ein späteres Auffinden der Dateien.

Normalerweise sucht sich Jumli automatisch den richtigen Pfad für die Compiler- bzw. Debugger-Befehle. Falls es bei Ihnen dennoch zu Problemen kommt, möchten wir Ihnen unter den nachfolgenden Links beim Lösen des Problems behilflich sein.

  • So setzt man die Pfade bei WindowsNT Systemen.

  • So setzt man die Pfade bei Windows 9x Systemen.

Nicht notwendig, aber sehr sinnvoll ist auch eine Installation der Java-Dokumentation. Sie besteht aus HTML-Seiten und beschreibt jede Java-Klasse der JDK sehr gut und umfangreich. Auch diese können Sie kostenlos von Sun downloaden
Für den Inhalt dieser Seiten ist Jumli.de nicht verantwortlich! Nach dem Download erhalten Sie eine Datei, die sich "j2sdk1_3_0-doc.zip" oder ähnlich nennt. Dies ist ein komprimiertes Archiv und muss zuerst noch mit einem Entpacker "entpackt" werden. Beachten Sie bitte, dass es sinnvoll ist, die Dokumentation ins gleiche Verzeichnis wie Ihr SDK (also zum Beispiel c:\jdk1.4) zu entpacken. Durch das Entpacken der Hilfe entsteht dann das neue Verzeichnis mit der Dokumenttion, z.B. c:\jdk1.4\docs. Diese Dokumentation benötigt ca. 100MB freien Speicherplatz auf Ihrer Festplatte.



Die Installation des .NET-SDK


Für die Programmierung in C# mit Jumli ist die Installation des SDK, also des "Software Development Kit " notwendig. Dieses wird Ihnen auf der Hompage von Microsoft .NET zum kostenlosen Download angeboten. Auch hier ist Jumli.de für den Inhalt dieser Seiten nicht verantwortlich!

Der Ablauf der Installation entspricht der des JDK von Sun.
Auch diesmal empfehlen wir das Ablegen der Dateien selbsterklärenden Ordnern wie z.B. c:\sdk oder c:\Programme\sdk.



Die Installationen für C++

Für die Programmierung mit C++ benötigen Sie mehrere Installationen. Eine genaue Anleitung finden Sie hier.

Nun sollten eigentlich die ersten Hürden genommen sein, und Jumli sollte so laufen wie Sie und wir uns das vorstellen. Wenn jedoch noch irgendetwas unklar sein sollte oder irgendetwas falsch läuft (ihr PC hat etwas falsch verstanden und sucht jetzt das Output-Device "Kaffeemaschine"), dann schreiben sie einfach eine E-mail an unseren Support.
