The Jumli Installation

Installing Jumli is as easy as falling of a log. After downloading Jumli from www.jumli.de you receive a "exe" File for Windows (or a comparable set of files if you downloaded the Linux version). With one doubleclick on "setup.exe" you start the installation. Just follow the installation hints now. If the installation was completed sucessfully there will be a Jumli-Icon in the start menu. A click on this Icon will start your Jumli.



The Installation of the Java-SDK


Before you can start programming in Java with your Jumli, you have to install the J2se-sdk (Java 2 Platform Standard Edition Software Developement Kit , briefly called JDK). The latest JDK version is offered to you for free on the Hompage of Sun Microsystems. Please notice that Jumli.de is not responsible for the contents of any website mentioned here!

You will receive a "exe" File, which will help you installing the JDK on your computer. We recommend you to install the JDK to a directory just as "c:\jdk1.4 or c:\Programme\jdk1.4". This will make it easier for you to work with the JDK later on.

Normally Jumli should automatically set the correct paths for compiler and/or debugger instructions.. However, if you might have problems setting the paths just have a look at the following links:

  • Setting paths for WindowsNT systems.

  • Setting paths for Windows 9x systems.

Not necessarily, but very meaningfully is also an installation of the Java documentation. It consists of HTML sites and describes each Java class of the JDK very well and detailed. Also these cannot do you free of charge of Sun downloaden for contents of these sides are responsible to Jumli.de! After the Download you receive a file, which calls itself "j2sdk1_3_0-doc.zip" or similarly. This is compressed archives and must first still with a Entpacker "unpacked" to become. Note please that it is meaningful to unpack the documentation in the same listing as your SDK (thus for example c:\jdk1.4). From unpacking the assistance results then the new listing with the Dokumenttion, e.g. to c:\jdk1.4\docs. This documentation needs approx. 100MB free storage location on your non removable disk.



The Installation of the .NET-SDK


For programming in C # with Jumli the installation of the SDK is, thus "software development kit" necessary. This is offered to you on the Hompage by Microsoft NET for the free Download. Jumli.de is not responsible for contents of these sides also here!

The expiration of the installation corresponds to that of the JDK von Sun. Also this time we recommend the placing of the files to self-describing files e.g. c:\sdk or c:\Programme\sdk.
