WindowsNT – path settings



It is important that the system path is set correctly by you. If you have a Windows NT system you have to click:

start > attitudes > system attitudes > system > environment

Here you have to select "Path". In the field "value" must stand "c:\jdk1.4\bin" (at the best at the end of the whole list). If you have installed the JDK under c:\jdk1.4, otherwise evenly the appropriate path has to be used. The path should not contain blanks. At the end you MUST press the button 'set' otherwise the new data won't be assumed!

Some computers also need that the variable "Classpath" is set. The value of this variable must be a single point.
