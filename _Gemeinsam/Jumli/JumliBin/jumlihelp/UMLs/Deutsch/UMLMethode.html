Was ist eine Methode?



Was ist eine Methode eigentlich?

Eine Methode (Nachricht) besteht aus einem Namen und einer Liste von Parametern. Methoden definieren Verhaltensweisen und Funktionalitäten eines Objekts. Falls eine Methode als öffentlich (public) deklariert wurde, kann ein anderes Objekt diese Funktionalität aufrufen und erhält einen Rückgabewert zurück.

Eine Methode trägt eine innerhalb einer Klassendefinition eindeutige Signatur, die sich aus Methodenname, Parametern und Rückgabewert zusammensetzt. Jedes Objekt versteht genau die Nachrichten, zu denen es eine entsprechende Methode besitzt.


Wie sieht so eine Methode denn in Jumli aus?

Wie bei den Attributen werden die Zugriffsrechte von Methoden durch Symbole dargestellt.

Über das Kontextmenue kann man einstellen, ob Jumli auch die Parameter der Methode zeichnen soll.
