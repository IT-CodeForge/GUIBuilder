Was ist ein Attribut?



Was ist ein Attribut eigentlich?

Ein Attribut wird durch seinen Namen beschrieben, es wird durch einen Datentyp sowie einen Initialwert (wenn gewünscht) definiert.
Der Datentyp ist programmiersprachenabhängig, in Java kann dies z.B. eine Klasse, ein zusammengesetzter oder elementarer Datentyp sein. Beispiele hierfür sind: int als Datentyp für Zahlen oder boolean für Wahrheitswerte.
Statische Attribute sind nicht Attribut eines einzelnen Objekts, sondern der Klasse, d.h. alle Objekte greifen auf den selben Wert zu. Wird dieser von einem Objekt verändert, dann sehen alle anderen Objekte ebenfalls diesen neuen Wert.
Je nach Programmiersprache können Zugriffsrechte auf Attribute vergeben werden, bei Java sind dies public, protected, private und unspecified.


Wie sieht so ein Attribut denn in Jumli aus?

Attributnamen sind ohne Sonderzeichen oder Leerstellen einzugeben.

Jumli verwendet zur Kennzeichnung von Zugriffsrechten Symbole. Diese sehen Sie links angezeigt.
