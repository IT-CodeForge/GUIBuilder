Was ist ein Package?



Was ist ein Package eigentlich?

Packages werden aufgrund logischer oder physischer Zusammenhänge gebildet, vorhandene Bibliotheken, Untersysteme und Schnittstellen bilden jeweils eigene Packages. Packages dienen dazu, einen besseren Überblick über das Model zu schaffen. Packages können hierarchisch gegliedert werden.

Wie sieht so ein Package denn in Jumli aus?

Ein Package wird in Form eines Aktenregisters dargestellt. Der Name steht innerhalb des großen Rechtecks.
