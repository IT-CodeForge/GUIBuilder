Was ist eine Entscheidung?



Was ist eine Entscheidung eigentlich?

Eine Entscheidung ist im Aktivity-Diagramm an den Stellen zu finden, an denen im System eine Bedingung mit Boolschem Ausdruck steht. Die Ablauf wird in der Richtung fortgeführt, für die die Bedingung erfüllt ist.

Wie sieht so eine Entscheidung denn in Jumli aus?

Entscheidungen werden in Jumli durch eine Raute dargestellt.
