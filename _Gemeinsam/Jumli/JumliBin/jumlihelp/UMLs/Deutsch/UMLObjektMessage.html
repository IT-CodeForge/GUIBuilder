Was ist eine Objekt-Nachricht?



Was ist eine Objekt-Nachricht eigentlich?

Eine Objekt-Nachricht ist eine auf Objekte im Sequenz- oder Kollaborationsdiagramm anwendbare Methode. Diese Objekt-Nachrichten werden in der Regel von einem Objekt an ein anderes Objekt geschickt. Start- und Zielobjekt können allerdings auch identisch sein.

Wie sieht so eine Objekt-Nachricht denn in Jumli aus?

Eine Objekt-Nachricht in Jumli wird durch einen Pfeil von einem Objekt zum Nächsten dargestellt. Außerdem werden Name der Methode, Zugriffmodifizierer und Rückgabewert angezeigt. Durch einen grauen Balken wird die Laufzeit der Methode im System auf der Zeitachse der Objekte dargestellt.
