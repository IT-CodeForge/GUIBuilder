Der Package Dialog




    1. Geben Sie hier den Namen des Packages ein. Er darf keine Leerstelle und kein Sonderzeichen enthalten.

    2. Hier können Sie einen Stereotype nennen.

    3. Um ein neues Diagramm zu erzeugen, betätigen Sie diesen Button.

    4. Hiermit kehren Sie zum Hauptdiagramm zurück.

    5. Um das aktuelle Package zu entfernen. Wenn Sie 'Entf' gedrückt haben, müssen Sie anschließend OK drücken, um das Entfernen auszuführen. Nach einer nochmaligen Abfrage können Sie das Package entfernen. Dies geht aber nur, wenn alle Klassen und Diagramme vorher gelöscht wurden.

    6. Hier kommen Sie in den Doku-Dialog.
