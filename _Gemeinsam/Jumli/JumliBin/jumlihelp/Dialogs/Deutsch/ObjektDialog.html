Der Objekt Dialog





1. Geben Sie hier den Namen des Objektes ein.

2. Nachdem Sie die Klasse ausgewählt haben zu der das Objekt gehört, können Sie sie hier sehen.

3. Hier sind alle möglichen Klassen oder Use Cases (bzw. Actor) enthalten, von denen Ihr Objekt erzeugt werden kann. Mit einem Doppelklick wird die Klasse im vorhergehenden Feld (2.) festgehalten.

4. Geben Sie hier einen Zustand ein.

5. Um das aktuelle Objekt zu entfernen. Wenn Sie 'Entf' gedrückt haben, müssen Sie anschließend OK klicken, um das Entfernen auszuführen. Nach einer nochmaligen Abfrage wird das Objekt gelöscht.

6. Hiermit gelangen Sie in den Doku-Dialog.

7. Hier können Sie eine neue Klasse erzeugen.
