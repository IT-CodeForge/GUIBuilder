Die Jumli Dialoge

Im Jumli gibt es zur Erstellung des UML-Modells viele verschiedene Dialoge. Da es verschiedene Möglichkeiten gibt, diese Dialoge einzustellen, wird hier erklärt, was in den einzelnen Dialogen einzustellen ist.

Im Jumli gibt es die Optionen-Dialoge:
  • Jumli-Optionen Dialog
  • JDK-Optionen Dialog
  • SDK-Optionen Dialog
Und die UML-Dialoge:
  • Actor Dialog
  • Aktivitäten Dialog
  • Assoziationen Dialog
  • Attribut Dialog
  • Beziehungs-Dialog
  • Dependency Dialog
  • Diagramm Dialog
  • Doku Dialog
  • Entscheidung Dialog
  • Klassen Dialog
  • Methoden Dialog

  • Objekt Dialog
  • Objekt-Link Dialog
  • Objekt-Nachrichten Dialog
