Der JDK-Optionen Dialog




    1. Hier muss der Pfad stehen, auf dem das JDK zu finden ist. Für gewönlich sollte Jumli diesen von alleine finden.

    2. Damit können Sie den Pfad für das JDK von Hand einstellen.

    3. Der Pfad für die Docs muss hier stehen. Dieser muss von Hand eingegeben werden.

    4. Hiermit können Sie den Pfad für die Docs verändern.

    5. Hier wird die Seite der JDK Klassen-Übersicht angegeben.

    6. Die Seite der Übersicht über die JDK Packages steht hier.

    7. Das Unterverzeichnis wird hier eingegeben.
