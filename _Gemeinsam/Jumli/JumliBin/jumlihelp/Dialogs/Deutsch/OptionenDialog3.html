Der SDK-Optionen Dialog




    1. Hier muss der Pfad stehen, auf dem das SDK zu finden ist. Für gewönlich sollte Jumli diesen von alleine finden.

    2. Damit können Sie den Pfad für das SDK von Hand einstellen.

    3. Hier ist der Pfad für den SDK-Compiler aufgeführt. Auch diesen sollte Jumli selbst erkennen. Da der Pfad unter Umständen jedoch nicht ermittelt wird (v.a., wenn Sie das Framework Version 1.1 nutzen; hier wird der Compiler und die Dokumenation in verschiedenen Ordnern abgelegt), können Sie den Pfad des Compilers selbst angeben. Beim Framework Version 1.0 befindet sich dieser für gewöhnlich im Verzeichnis des Frameworks, bei der Version 1.1 für gewöhnlich in einem Unterverzeichns (Microsoft.NET\...) des Windows-Ordners.

    4. Hier kann der Pfad des Compilers von Hand verändert werden.
