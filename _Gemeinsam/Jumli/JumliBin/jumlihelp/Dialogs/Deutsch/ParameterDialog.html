Der Parameter Dialog




    1. Geben Sie hier den Namen des Parameters an. Er darf keine Leerstelle und kein Sonderzeichen enthalten.

    2. Geben Sie hier den Typ des Parameters an, z.B. int, boolean, String ...

    3. In C++ kan eine Default-Wert des Parameters angegeben werden.

    4. Hier können Sie die Vorbelegung des Parameters angeben.

    5. Hier kommen Sie in den Doku-Dialog.
