Der Objekt-Nachrichten Dialog





1. Haben Sie bei 5. eine Methode mit Doppelklick ausgesucht, steht Sie hier.

2. Hier steht der Sender der Objekt-Nachricht.

3. Der Empfänger der Objekt-Nachricht steht in diesem Feld.

4. Nummer

5. Die möglichen Methoden, die für diese Objekt-Nachricht in Frage kommen, finden Sie hier.

6. Aber Sie können auch erst in diesem Dialog eine Methode erstellen, die Sie anschließend in die zugehörige Klasse übernehmen können.

7. Um die aktuelle Objekt-Nachricht zu entfernen. Wenn Sie 'Entf' gedrückt haben, müssen Sie anschließend OK klicken, um das Entfernen auszuführen. Nach einer nochmaligen Abfrage wird die Objekt-Nachricht gelöscht.

8. Hiermit gelangen Sie in den Doku-Dialog.
