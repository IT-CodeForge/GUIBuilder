Method Dialog




    1. Enter the name of the method here.

    2. Enter the return value of the method here.

    3. If you need throws enter them here.

    4. Here you can define the access of the method. You have the choice between:

      private: The method is visible and well-known only within the class.
      protected: The method can be called from classes within the packages and classes derived from.
      public: All objects, which know the associated class, can call the method.
      internal: This option is available only in C #.
      protected internal: Also this right of access can be used only in C #.
      unspecified: The method is well-known only in the package.



5. Here you can define the characteristics of a method. There is the selection between:

    abstract: This method must be overwritten by a derived class.
    final: The method can't be overwritten by another class (only possible in Java).
    static: The method isn't individually create for each object, but exists only once for the whole class. All objects of this class have access to this method, i.e. if the method is changed by one, appears them for all changed. The access is also possible, if still no object of this class exists.

    native: This characteristic can be used only in Java.
    synchronized: A class can be changed by several threads "at the same time". If one sets the characteristic synchronized, then it is guaranteed that only a thread changes for each time unit the class. This is possible only in Java.

    explicit: This is possible only in Java.
    const: In C++ a method can be defined const.
    virtual: This is possible only in Java.
    implicit, new, override, ist delegate: This is possible only in C#.



6. The parameters of the method are listed here.

7. If you press the button 'New' you come into the Parameter Dialog .

8. With 'Del' you can remove the marked parameter, then the parameter is deleted after a repeated inquiry.

9. Here you leave the dialog and store the entered data.

10. To leave the dialog without storing click 'Cancel'.

11. Here you come into the Docu-Dialog.

12. Here you come into the dialogue help, in whom you are for the moment.
