SDK-Options Dialog




    1. Here you have to enter the path on which the SDK is to be found. Normally Jumli should find this path automatically.

    2. Choose the path here.

    3. Here you have to enter the path on which the Compiler is to be found. Normally Jumli should find this path from alone.

    4. Choose the path here.

    5. Here you leave the dialog and store the entered data.

    6. To leave the dialog without storing click 'Cancel'.

    7. Here you come into the dialogue help, in whom you are for the moment.
