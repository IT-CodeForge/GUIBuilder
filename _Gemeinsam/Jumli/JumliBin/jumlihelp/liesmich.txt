Jumli 1.4
---------

Willkommen zu Jumli 1.4! 
Mit Jumli steht Ihnen ein Tool zur Verfügung, das sowohl einen UML-Editor als auch einen Quellcode-Editor für Java, C# und C++ enthält. Obwohl Jumli ursprünglich nur als Schulungstool gedacht war, besitzt es eine Reihe von nützlichen Features, die es auch erfahrenen Programmierern erleichtern, die verschiedenen Aspekte der objektorientierten Analyse, Design und Programmierung in die Praxis umzusetzen.

Mit Jumli ist es möglich, UML-Diagramme zu erstellen (z.B. Use-Case-Diagramm, Klassendiagramm, Aktivitätsdiagramm, Sequenzdiagramm, Kollaborationsdiagramm, Komponentendiagramm etc.) und aus diesem Modell Quellcode zu generieren. Weiterhin kann mit Jumli bereits vorhandender Quellcode analysiert und als UML-Klassendiagramm dargestellt werden, so dass auch Roundtrip-Engeneering ermöglicht wird.



Neue Features in 1.4:
---------------------------

* Icon Editor
* XML- / XMI- Export
* HTML-Export
* Anzeige von UML-konformen Icons in Diagrammen
* Trennung von Dokumentations -und Nachrichtenfenster
* Autocomplete
* Erweiterte Funktionen im Projektbaum
* Generierung von Datenbankklassen ( nur in C++ )
* Suchen im Model
* Kopieren von Klassen, Methoden und Attributen
* Verbessertes Diagrammhandling
* Angabe von Stereotyps bei Use Cases, Actors und Aktivities möglich
* Verbesserte Editor-Funktionen: Bookmarks, Folding, Zeilennummern usw.
